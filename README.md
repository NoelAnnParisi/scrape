# NYT Scraper
This Node app scrapes the most popular news headlines and allows users to comment on them. User's can view previsouly scraped headlines or the most recent. 

## Technologies Used: 
* Ecmascript 2015
* Cheerio
* Express.js
* Node.js
* Mongoose
* Handlebars.js
* Tachyons.io 
* HTML5/CSS3
* Deployed on Heroku
